## 2. 文本安全检测

### 接口信息

- **路径**: `/api/aigc/security/check/text`
- **方法**: `POST`
- **Content-Type**: `application/json`
- **说明**: 检测文本内容是否包含敏感、违规信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|------|------|--------|
| tenantId | integer | 是 | 租户ID | -173 |
| robotCode | string | 是 | 应用代码 | "aiguangct2p" |
| robotType | string | 是 | 应用类型 | "jaingsuhaobai-telecom" |
| content | string | 是 | 待检测的文本内容（UTF-8编码） | "这是需要检测的文本" |
| contentType | string | 是 | 内容来源类型 | "userQuery" 或 "modelResponse" |

**contentType 取值说明**:
- `userQuery`: 用户输入的内容
- `modelResponse`: AI模型生成的内容

### 请求示例

#### cURL

```bash
curl -X POST http://YOUR_PROXY_SERVER_IP:5000/api/aigc/security/check/text \
  -H "Content-Type: application/json" \
  -d '{
    "tenantId": -173,
    "robotCode": "aiguangct2p",
    "robotType": "jaingsuhaobai-telecom",
    "content": "这里是需要检测的敏感测试文本",
    "contentType": "userQuery"
  }'
```

#### Python


#### Java

```java
import java.net.http.*;
import java.net.URI;
import com.google.gson.Gson;
import com.google.gson.JsonObject;

public class TextSecurityCheck {
    public static void main(String[] args) throws Exception {
        String url = "http://YOUR_PROXY_SERVER_IP:5000/api/aigc/security/check/text";

        JsonObject payload = new JsonObject();
        payload.addProperty("tenantId", -173);
        payload.addProperty("robotCode", "aiguangct2p");
        payload.addProperty("robotType", "jaingsuhaobai-telecom");
        payload.addProperty("content", "这里是需要检测的敏感测试文本");
        payload.addProperty("contentType", "userQuery");

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create(url))
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(new Gson().toJson(payload)))
            .build();

        HttpResponse<String> response = client.send(request, HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## 3. 图片安全检测

### 接口信息

- **路径**: `/api/aigc/security/check/picture`
- **方法**: `POST`
- **Content-Type**: `application/json`
- **说明**: 检测图片内容是否包含敏感、违规信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|------|------|--------|
| tenantId | integer | 是 | 租户ID | -173 |
| robotCode | string | 是 | 应用代码 | "aiguangct2p" |
| robotType | string | 是 | 应用类型 | "jaingsuhaobai-telecom" |
| content | string | 是 | 图片的Base64编码字符串 | "iVBORw0KGgoAAAANS..." |
| contentType | string | 是 | 内容来源类型 | "userQuery" 或 "modelResponse" |

> **注意**: content 字段需要传入图片文件的 Base64 编码字符串（不包含 `data:image/png;base64,` 前缀）

### 图片转Base64示例

#### Python

```python
import base64

def image_to_base64(image_path):
    """将图片文件转换为Base64字符串"""
    with open(image_path, "rb") as image_file:
        return base64.b64encode(image_file.read()).decode('utf-8')

# 使用示例
base64_str = image_to_base64('/path/to/image.jpg')
print(base64_str)
```

#### JavaScript (Node.js)

```javascript
const fs = require('fs');

function imageToBase64(imagePath) {
  const imageBuffer = fs.readFileSync(imagePath);
  return imageBuffer.toString('base64');
}

// 使用示例
const base64Str = imageToBase64('/path/to/image.jpg');
console.log(base64Str);
```

#### Bash

```bash
base64 -w 0 /path/to/image.jpg
```

#### Python

```python
import requests
import base64
import json

def check_image_security(image_path):
    """检测图片安全性"""
    # 读取图片并转换为Base64
    with open(image_path, "rb") as image_file:
        base64_str = base64.b64encode(image_file.read()).decode('utf-8')

    # 构造请求
    url = "http://YOUR_PROXY_SERVER_IP:5000/api/aigc/security/check/picture"
    payload = {
        "tenantId": -173,
        "robotCode": "aiguangct2p",
        "robotType": "jaingsuhaobai-telecom",
        "content": base64_str,
        "contentType": "userQuery"
    }

    # 发送请求
    response = requests.post(url, json=payload, timeout=30)
    result = response.json()

    return result

# 使用示例
result = check_image_security('/path/to/image.jpg')
print(json.dumps(result, ensure_ascii=False, indent=2))
```

#### JavaScript (Node.js)

```javascript
const axios = require('axios');
const fs = require('fs');

async function checkImageSecurity(imagePath) {
  // 读取图片并转换为Base64
  const imageBuffer = fs.readFileSync(imagePath);
  const base64Str = imageBuffer.toString('base64');

  // 构造请求
  const url = 'http://YOUR_PROXY_SERVER_IP:5000/api/aigc/security/check/picture';
  const payload = {
    tenantId: -173,
    robotCode: 'aiguangct2p',
    robotType: 'jaingsuhaobai-telecom',
    content: base64Str,
    contentType: 'userQuery'
  };

  try {
    const response = await axios.post(url, payload);
    console.log(JSON.stringify(response.data, null, 2));
    return response.data;
  } catch (error) {
    console.error('Error:', error.message);
  }
}

// 使用示例
checkImageSecurity('/path/to/image.jpg');
```

## 状态码说明

| HTTP状态码 | code | 说明 |
|-----------|------|------|
| 200 | 10000 | 请求成功 |
| 400 | 40000 | 请求参数错误（缺少必填字段等） |
| 500 | 50000 | 服务器内部错误或转发失败 |
