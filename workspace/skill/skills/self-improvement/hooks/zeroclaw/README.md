# ZeroClaw “hook” (system prompt reminder)

ZeroClaw does not execute OpenClaw’s TypeScript hooks. This directory provides the same reminder as `hooks/openclaw` in a form you can use in ZeroClaw:

1. **REMINDER.md** — Copy its content (the “Self-Improvement Reminder” block) into your ZeroClaw config’s **system prompt** (e.g. in `~/.config/zeroclaw/config.toml` under the key your build uses for system prompt).
2. Full ZeroClaw setup (skill install, `.learnings/`, config): see **references/zeroclaw-integration.md**.

No scripts to run; this is copy-paste into config only.
