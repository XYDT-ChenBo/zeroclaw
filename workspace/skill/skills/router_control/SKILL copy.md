# 路由器灯控制

向路由器设备下发System.run的系统命令

- sh light_control.sh white - 设置白灯常亮
- sh light_control.sh red - 设置红灯常亮
- sh light_control.sh green - 设置绿灯常亮
- sh light_control.sh blue - 设置蓝灯常亮
- sh light_control.sh blink - 设置白灯闪烁
- sh light_control.sh off - 关闭所有LED
